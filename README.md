# control flow
- if, elif, else
- Conditional statements are used to control the flow of our program

### examples:
- if:  
  ```
  if age >= 15:
    print("Thank you, please proceed to your purchase")
  ```
  
- elif: 
    ```
  elif age < 15:
    print("You're too young to watch this movie")
  ```  
  
- else:
    ```
  else:
    print("something went wrong")
  ```
